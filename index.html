Explanation

Type in the English names of the cards you want to use above. You will get a list of all of the Tier "1-3" decks sorted by number of matches. As an example, all cards from Alpha edition are given. Partial matches ("Bolt","Cradle") and typos ("Thoughtsieze", "Uwezama's Jitte") are automatically corrected.

Motivation

Do you have some cards lying around that you can't find a home for? Sites like EDHREC show you decks for a single card but I haven't found any tool that maximizes the use of multiple cards. Also, I wanted to only include competitive decks.

Questions and Contributions

This is a quick hack developed on GitHub. Feel free to contribute a pull request or to open an issue to report bugs, ask questions or propose additions.

Known Bugs

Tier 3 isn't complete and contains some invalid decks, I still included it for the sake of completeness. Because I couldn't get the tappedout API to work, I had to rely on quick and dirty Web scraping, which means that some cards may end up in decks when they are only discussed in the comments.

Requirements

The code only uses JavaScript, but some ES5/ES6 features. Any recent browser should work fine, but I don't think it will work on Internet Explorer.
